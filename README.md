### Building with maturin
` maturin build -f -r -b cffi ` 

### Running in python

```python
import __main__
import cv2
import numpy

image: numpy.ndarray = cv2.imread(r"/images/G1.jpg", cv2.IMREAD_COLOR)
image_wave = py_lens.apply_lens(image, cx=1, cy=0.5, U=100)
cv2.imshow('G1-1-05-100.jpeg', image_wave)
cv2.waitKey()
```
